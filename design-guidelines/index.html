Design Guidelines

Design guidelines

Typography

  • Use clear, readable sans-serif type for body content
  • Keep headings prominent and concise
  • Maintain strong contrast for accessibility

Color

  • Favor calm, modern neutrals with a strong accent color
  • Use color intentionally to guide attention and create hierarchy

Spacing

  • Keep spacing consistent across sections and cards
  • Use generous padding on mobile to improve readability

Interaction

  • Make buttons and links obvious and easy to tap
  • Support keyboard navigation and clear focus states
